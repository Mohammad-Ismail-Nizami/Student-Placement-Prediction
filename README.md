# 🎓 Student Placement Prediction using Machine Learning

Predict a student's placement probability using Machine Learning and an interactive Streamlit web application.

## 🌐 Live Demo

🔗 https://student-placement-prediction-ml.streamlit.app/

---

## 📌 Overview

This project predicts whether a student is likely to be placed based on academic performance, technical skills, practical experience, and soft skills.

The project demonstrates a complete end-to-end Machine Learning workflow—from data preprocessing and feature engineering to model deployment using Streamlit.

---

## ✨ Features

- Interactive Streamlit web application
- Placement probability prediction
- Personalized recommendations
- Readiness score
- Probability visualization
- Feature contribution analysis
- Logistic Regression Pipeline
- Responsive user interface
